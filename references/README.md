# 文献资料（References）

把项目需要参考的文献统一放在此目录，便于管理与在论文/报告中引用。

## 建议的文件命名

优先使用：

`作者-年份-标题关键词.pdf`

示例：

- `Goodfellow-2014-GAN.pdf`
- `He-2016-ResNet.pdf`
- `Smith-2022-RobotGraspingSurvey.pdf`

如果是多作者可用第一作者 + `EtAl`：

`FirstAuthorEtAl-年份-标题关键词.pdf`

## 可放的文件类型

- 论文：`.pdf`
- 预印本/草稿：`.pdf`/`.docx`
- 读书笔记：`.md`/`.txt`
- 引文库：`.bib`（BibTeX）/`.ris`

## 给我处理文献时，建议你在消息里说明

1. 想要的输出：摘要/要点/方法对比/相关工作综述/可直接引用的中文表述等
2. 目标章节：如“相关工作”“方法”“实验”
3. 引用风格：GB/T 7714、APA、IEEE 或你学校模板
