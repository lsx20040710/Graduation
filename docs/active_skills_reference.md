# 当前项目技能保留说明

## 作用

这份说明用于记录当前毕业设计项目建议保留的 Codex 技能集合，避免后续整理本机 `C:\Users\LSX\.codex\skills` 时反复判断。

## 建议保留的活动技能

- `.system`
- `codex-primary-runtime`
- `0-autoresearch-skill`
- `doc`
- `pdf`
- `spreadsheet`
- `academic-plotting`
- `ml-paper-writing`
- `systems-paper-writing`
- `presenting-conference-talks`
- `imagegen`

## 建议移出活动集合的技能类型

以下技能通常不需要为当前项目常驻保留：

- 大模型训练与微调类
- 模型量化与压缩类
- 大规模推理服务类
- 云平台、分布式训练和集群运维类
- 与当前海参采捕软体机器人主线无直接关系的通用前沿研究技能

## 判断标准

如果一个技能不能直接帮助当前四类工作，就不建议继续常驻：

1. 视觉检测与误差提取
2. 六舵机绳驱控制实验
3. 论文写作与图表整理
4. 答辩材料输出

- `build/`、`install/`、`log/`、`__pycache__/`、`src/Identification_code/runs/` 的位置保持不变，本说明不参与目录清理。
